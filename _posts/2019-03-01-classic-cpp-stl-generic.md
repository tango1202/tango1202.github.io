# 타입 기반 프로그래밍

타입 기반 프로그래밍 : Type으로 동작 방식 결정
문제 해결을 위한 알고리즘 일반화
여러 타입과 자료구조에 대해 동작할 수 있도록 알고리즘을 매개 변수화
공통의 상황/처리방식에 대한 추상화

# 개체 지향과 일반화

개체지향 : 데이터와 처리 방식의 응집
클래스마다 다르게 구현

일반화 : 데이터와 처리방식의 분리
클래스에 따라 동일하게 구현. 동일하지 않으면 특수화
